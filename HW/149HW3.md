## 1. Utility Class – `System.Math`

ลิงก์: [https://learn.microsoft.com/en-us/dotnet/api/system.math?view=net-10.0](https://learn.microsoft.com/en-us/dotnet/api/system.math?view=net-10.0)

### Definition

เหตุผลที่ `Math` จัดเป็น Utility Class เพราะเป็น **static class** ที่รวบรวมฟังก์ชันและค่าคงที่ทางคณิตศาสตร์ไว้ให้เรียกใช้งานโดยตรง ไม่จำเป็นต้องสร้างอ็อบเจ็กต์ (instance) และไม่ได้เก็บข้อมูลสถานะ (state) ภายในตัวเอง จึงทำหน้าที่เป็นเพียงเครื่องมือสำหรับการคำนวณล้วน ๆ

### Examples

```csharp
using System;


class Program
{
static void Main()
{
double radius = 7;


double area = Math.PI * Math.Pow(radius, 2);
double rounded = Math.Round(area, 2);


Console.WriteLine("Area: " + area);
Console.WriteLine("Rounded Area: " + rounded);
}
}
```

### Analysis

* ใช้ `Math.PI`, `Math.Pow()`, `Math.Round()` ซึ่งเป็นสมาชิกของ static class Math
* เรียกใช้งานเมธอดโดยตรงผ่านชื่อคลาส ไม่ต้องสร้าง instance
* ไม่มีการเก็บข้อมูลหรือ state ภายในคลาส
* ทำหน้าที่เป็นเครื่องมือคำนวณทางคณิตศาสตร์ล้วน ๆ
* สอดคล้องกับแนวคิด Utility class (ใช้งานเป็นฟังก์ชันสำเร็จรูป)

## 2. Mutable Class – `System.Text.StringBuilder`

ลิงก์: [https://learn.microsoft.com/en-us/dotnet/api/system.text.stringbuilder?view=net-10.0](https://learn.microsoft.com/en-us/dotnet/api/system.text.stringbuilder?view=net-10.0)

### Definition

`StringBuilder` ถูกจัดเป็น Mutable Class เนื่องจากเอกสารระบุว่าเป็น **“mutable string of characters”** หมายความว่า ข้อมูลภายในอ็อบเจ็กต์เดียวกันสามารถถูกแก้ไข เปลี่ยน เพิ่ม หรือลบได้โดยตรง ผ่านเมธอดอย่าง `Append`, `Insert`, `Remove`, `Replace` ซึ่งทำให้ state ภายในเปลี่ยนแปลงตลอดเวลา

### Examples

```csharp
using System;
using System.Text;


class Program
{
static void Main()
{
StringBuilder log = new StringBuilder();


log.AppendLine("Start Program");
log.AppendLine("Loading data...");
log.AppendLine("Process completed");


log.Insert(0, "[SYSTEM LOG]\n");


Console.WriteLine(log.ToString());
}
}
```

### Analysis

* สร้าง object StringBuilder เพียงครั้งเดียว
* ใช้ `AppendLine()` เพิ่มข้อมูลต่อท้ายข้อความเดิม
* ใช้ `Insert()` แทรกข้อความเข้าไปในตำแหน่งเดิมของ object
* ข้อมูลถูกแก้ไขใน instance เดิมทั้งหมด
* แสดงลักษณะ mutable อย่างชัดเจน (state เปลี่ยนแปลงได้)

## 3. Immutable Class – `System.String`

ลิงก์: [https://learn.microsoft.com/en-us/dotnet/api/system.string?view=net-10.0](https://learn.microsoft.com/en-us/dotnet/api/system.string?view=net-10.0)

### Definition

`String` เป็นตัวอย่างคลาสแบบ Immutable อย่างชัดเจน กล่าวคือค่าภายในไม่สามารถแก้ไขได้โดยตรง การดำเนินการที่ดูเหมือนเป็นการเปลี่ยนข้อความ เช่น การต่อสตริง แท้จริงแล้วคือการสร้างอ็อบเจ็กต์ `string` ใหม่ขึ้นมาแทนที่ของเดิม โดยที่ object เดิมยังคงค่าเดิมอยู่

### Examples

```csharp
using System;


class Program
{
static void Main()
{
string baseText = "C#";


string v1 = baseText;
string v2 = baseText + " Programming";
string v3 = baseText.Replace("#", " Sharp");


Console.WriteLine(v1); // C#
Console.WriteLine(v2); // C# Programming
Console.WriteLine(v3); // C Sharp
}
}
```

### Analysis

* baseText เป็น `string` ต้นฉบับที่ไม่ถูกเปลี่ยนค่า
* การต่อข้อความ (+) จะสร้าง string ใหม่ (v2)
* การใช้ `Replace()` ก็สร้าง object string ใหม่ (v3)
* ค่าของ baseText ยังคงเดิมตลอดการทำงาน
* แสดงคุณสมบัติ immutable อย่างชัดเจน (ไม่แก้ object เดิม)
